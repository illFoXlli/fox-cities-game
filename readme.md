# Cities Game

Проста графічна гра "Міста", створена на Java Swing.

## Запуск

У IntelliJ IDEA запустіть клас:

```text
com.fox.AppLauncher
```

## Правила гри

Гравець вводить назву міста українською мовою. Комп'ютер відповідає містом, яке починається на останню допустиму літеру міста гравця. Повторювати вже використані міста не можна.

Гравець перемагає, якщо комп'ютер не може знайти відповідне місто. Комп'ютер перемагає, якщо гравець натискає кнопку "Здатись" або вводить у поле слово "здаюсь".
